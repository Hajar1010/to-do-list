# To-Do List CLI App

A simple command-line to-do list manager where you can add, view, and complete tasks â€” all from your terminal. This app helps practice file I/O, loops, and menu-driven programs in Python.

---

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Saves tasks in a text file (`tasks.txt`)

---

## Skills Practiced

- File Handling
- Loops & Conditionals
- List Management
- User Input Handling

---

## How to Install and Run
-Make sure you have Python 3.x installed on your computer.
-Download or clone this repository to your local machine.
-Open a terminal or command prompt in the project folder.
##Run the program with the command:
   **```bash**
python to_do_list.py
**```**
